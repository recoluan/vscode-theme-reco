# Reco Theme for Visual Studio Code

This Theme as much as possible according to vuepress-theme-reco.

## Syntax Support

- Python
- Javascript
- TypeScript
- React, Vue
- HTML
- Markdown
- JSON
- CSS, SCSS
- Dockerfile
- Go
- More and more...
